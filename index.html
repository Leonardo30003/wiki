¿Qué son los datos abstractos?

Los datos abstractos son estructuras de datos que se definen por su comportamiento en lugar de su implementación. En otras palabras, son objetos que se utilizan para almacenar datos y realizar operaciones en ellos, pero que no especifican cómo se llevan a cabo estas operaciones. En lugar de eso, se describen las propiedades y características generales de la estructura de datos, y se dejan los detalles de la implementación para cada caso en particular.

h1

Lista

Una lista es una colección ordenada de elementos. Los elementos pueden ser de cualquier tipo de datos, y se accede a ellos mediante su posición dentro de la lista. Las operaciones comunes que se pueden realizar en una lista son: Agregar un elemento al final de la lista. Eliminar un elemento de la lista. Buscar un elemento en la lista. Obtener el número de elementos en la lista.

Pila

Una pila es una estructura de datos que se utiliza para almacenar elementos en la que el último elemento en ser insertado es el primero en ser eliminado. Las operaciones comunes que se pueden realizar en una pila son: Empujar un elemento a la pila (añadir un elemento al final de la pila). Sacar un elemento de la pila (eliminar el último elemento de la pila). Obtener el número de elementos en la pila.

Cola

Una cola es una estructura de datos que se utiliza para almacenar elementos en la que el primer elemento en ser insertado es el primero en ser eliminado. Las operaciones comunes que se pueden realizar en una cola son: Añadir un elemento a la cola (añadir un elemento al final de la cola). Eliminar un elemento de la cola (eliminar el primer elemento de la cola). Obtener el número de elementos en la cola.

Árbol

Un árbol es una estructura de datos jerárquica que se utiliza para representar datos en forma de árbol. El nodo superior se llama la raíz, y cada nodo puede tener varios hijos. Los nodos que no tienen hijos se llaman hojas. Las operaciones comunes que se pueden realizar en un árbol son: Insertar un nodo en el árbol. Eliminar un nodo del árbol. Buscar un nodo en el árbol.

Ejemplo de implementación de una lista en Python

class Lista:
def init(self):
self.items = []
def agregar(self, item):
self.items.append(item)
def eliminar(self, item):
self.items.remove(item)
def buscar(self, item):
return item in self.items
def obtener_tamaño(self):
return len(self.items)

Ejemplo de implementación de una pila en Python

class Pila:
def init(self):
self.items = []
def esta_vacia(self):
return self.items == []
def empujar(self, item):
self.items.append(item)
def sacar(self):
return self.items.pop()
def obtener_tamaño(self):
return len(self.items)
mi_pila = Pila()
print(mi_pila.esta_vacia()) # True
mi_pila.empujar(1)
mi_pila.empujar(2)
mi_pila.empujar(3)
print(mi_pila.esta_vacia()) # False
print(mi_pila.obtener_tamaño()) # 3
print(mi_pila.sacar()) # 3
print(mi_pila.sacar()) # 2
print(mi_pila.sacar()) # 1
print(mi_pila.esta_vacia()) # True

Ejemplo de implementación de una cola en Python

class Cola:
def init(self):
self.items = []
def esta_vacia(self):
return self.items == []
def encolar(self, item):
self.items.insert(0, item)
def desencolar(self):
return self.items.pop()
def obtener_tamaño(self):
return len(self.items)
mi_cola = Cola()
print(mi_cola.esta_vacia()) # True
mi_cola.encolar(1)
mi_cola.encolar(2)
mi_cola.encolar(3)
print(mi_cola.esta_vacia()) # False
print(mi_cola.obtener_tamaño()) # 3
print(mi_cola.desencolar()) # 1
print(mi_cola.desencolar()) # 2
print(mi_cola.desencolar()) # 3
print(mi_cola.esta_vacia()) # True
